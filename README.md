# Keysharp - Avalonia Hello World

A simple Hello World application built with Avalonia UI for C#.

## Running the Application

To run the application, use:

```bash
dotnet run
```

## Building

To build the application:

```bash
dotnet build
```

## Project Structure

- `Program.cs` - Application entry point
- `App.axaml` / `App.axaml.cs` - Application configuration and initialization
- `Views/MainWindow.axaml` / `Views/MainWindow.axaml.cs` - Main window UI and logic
- `Keysharp.csproj` - Project file with Avalonia dependencies

## Features

- Simple Hello World greeting
- Interactive button that changes text when clicked
- Modern Fluent theme styling
- Cross-platform support (Windows, macOS, Linux)
